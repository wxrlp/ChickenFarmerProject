A Java farming simulation game. The player controls a Chicken Farmer who navigates a tile-based world to mine ores for coins, till dirt, plant cabbages, and manage resources — all while fending off hostile bird enemies (Eagles, Magpies, Pigeons) and interacting with NPCs like Guard Bees and Scarecrows.

Key features:

Tile-based world — Grass, Dirt, Water, and Ore Vein tiles loaded from .map files
Inventory system — Tools (Bucket, Hoe, Jackhammer, Pole) with animated use actions, plus coin & food tracking
Enemy AI — Spawner-driven enemies with autonomous movement and player interaction
NPC ecosystem — BeeHives, Guard Bees, Scarecrows, and a helper system that targets the player
Sprite-based rendering — Animated player movement, directional sprites, and layered tile/entity rendering Modular architecture — Clean separation across world, player, inventory, entity, and UI packages
